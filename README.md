# LinguaNova 🌐

> 🏆 3rd Prize Winner — RenAIssance Hackathon

LinguaNova is an AI-powered language learning app built in 48 hours during a hackathon. 
It makes language learning immersive and interactive through GenAI-driven conversations, 
image-based vocabulary, and adaptive storytelling.

---

## ✨ Features

- 🎭 **Immersive Roleplay** — Conversational AI roleplay powered by Google Gemini
- 🖼️ **Image-Based Vocabulary** — Learn words visually using WorqHat's image AI
- 📖 **Story Mode** — Adaptive stories with multiple difficulty levels
- 🌍 **Real-Time Translation** — Instant translations using NLP techniques

---

## 🛠️ Tech Stack

- **Google Gemini** — Generative AI for roleplay and conversations
- **WorqHat API** — Image-based vocabulary generation
- **NLP** — Real-time translation
- **Python** — Core backend logic

---

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Gemini API Key
- WorqHat API Key

### Installation
```bash
git clone https://github.com/Jsn04/HackAIthon-Project.git
cd HackAIthon-Project
pip install -r requirements.txt
```

### Run the app
```bash
python app.py
```
